# 智慧课堂互动平台 (Smart Classroom Interactive Platform)

## 1. 项目简介

本项目是一个基于 Django 框架开发的智慧课堂互动平台。旨在通过现代化的Web技术，连接教师、学生和管理员，提供丰富的教学互动、课程管理、作业考试和数据分析功能，从而提升教学效率和学习体验。

## 2. 技术选型 (Technology Stack)

*   **后端:**
    *   **框架:** Django
    *   **API:** Django REST Framework (用于前后端分离)
    *   **实时通信:** Django Channels (用于实现WebSocket课堂互动)
    *   **认证:** Django内置认证 + djangorestframework-simplejwt (用于JWT令牌)
*   **前端:** (建议)
    *   **框架:** Vue.js 或 React (用于构建单页面应用)
    *   **UI库:** Element Plus 或 Ant Design Vue
    *   **图表:** ECharts 或 Chart.js
*   **数据库:** PostgreSQL / MySQL
*   **部署:** Docker, Nginx, Gunicorn

## 3. 功能模块设计

根据功能图，系统分为三大端：教师端、学生端和后台管理端。

### 3.1 教师端

*   **课程管理:** 创建和管理课程，上传/分发课程资料，发布课程公告。
*   **教学互动:**
    *   通过WebSocket实现实时签到、提问、投票。
    *   创建和管理课堂讨论区。
*   **作业与考试管理:**
    *   发布作业和试卷（支持选择题、填空题、简答题等）。
    *   设置截止日期，在线批改作业，自动统计分数。
*   **教学反馈:** 发布教学问卷，查看学生反馈统计。
*   **个人中心:** 管理个人信息，修改密码。

### 3.2 学生端

*   **学习课程:** 选择/加入课程，查看课程列表，下载课程资料。
*   **课堂互动:**
    *   参与实时签到、提问、投票和讨论。
*   **作业与考试:**
    *   在线完成并提交作业/考试。
    *   查看批改后的作业和分数。
*   **学习反馈:** 匿名或实名填写教学问卷和反馈。
*   **个人中心:** 查看个人信息，修改密码。

### 3.3 后台管理端

*   **用户与权限管理:** 管理教师、学生、管理员三类角色的账户和权限。
*   **系统管理:** 配置系统参数，例如学期设置。
*   **数据统计与分析:**
    *   可视化展示全校/课程的教学数据（如出勤率、作业完成率、课堂活跃度）。
    *   支持数据导出。
*   **系统维护:** 查看系统日志，进行数据备份与恢复。

## 4. 项目启动与安装

1.  克隆项目到本地: `git clone https://github.com/Keke2004/iclass.git`
2.  进入项目目录: `cd iclass`
3.  创建并激活Python虚拟环境。
4.  安装依赖: `pip install -r requirements.txt`
5.  配置数据库并执行迁移: `python manage.py migrate`
6.  启动开发服务器: `python manage.py runserver`
